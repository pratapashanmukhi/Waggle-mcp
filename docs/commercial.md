# Waggle Commercial Roadmap

`waggle-mcp` is the public Apache-2.0 core of Waggle.

## Waggle Core

Waggle Core is the open-source foundation:

- local-first memory graph
- SQLite-backed storage
- basic MCP server
- basic CLI workflows
- public GitHub repo
- public PyPI package

## Waggle Plus

Waggle Plus is planned as the paid layer on top of Waggle Core.

Status: `Coming soon`

Planned areas:

- advanced reranking and retrieval
- contradiction intelligence
- premium context-building workflows
- higher usage limits in official binaries
- team and enterprise features

Waggle Plus is not available yet, and no payment flow is live yet.

Until then, this repository and the `waggle-mcp` package represent the public
Waggle Core only.
